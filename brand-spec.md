# Brand Spec — صراط (Sirat)

## Product
Open-source Android application for digital wellness, accountability, and self-discipline.

## Color Palette (hex → OKLch)

| Token    | Hex       | OKLch                     | Usage                        |
|----------|-----------|---------------------------|------------------------------|
| `--bg`   | `#FBF7E3` | `oklch(96% 0.025 95)`     | Page background              |
| `--surface` | `#FFFFFF` | `oklch(100% 0 0)`      | Cards, modals, sections      |
| `--fg`   | `#216E5B` | `oklch(42% 0.10 175)`     | Headers, primary text        |
| `--muted`| `#5A7A72` | `oklch(52% 0.04 175)`     | Body text, secondary          |
| `--border`| `#E0DBD0` | `oklch(88% 0.015 90)`    | Dividers, borders             |
| `--accent-teal` | `#92D2C7` | `oklch(78% 0.08 185)` | UI elements, buttons, highlights |
| `--accent-green` | `#99C78F` | `oklch(75% 0.10 145)` | Secondary accents, badges    |
| `--accent-yellow` | `#FCDA7B` | `oklch(85% 0.12 85)` | Highlights, CTAs, emphasis   |

## Typography
- **Display**: Cairo (Google Fonts), 700–900 weight
- **Body**: Tajawal (Google Fonts), 400–700 weight
- **Monospace**: system-ui / fallback

## Logo
- File: `app_icon.png`
- Position: Hero section, prominently displayed
- Background: transparent (app icon with green-teal-yellow pastel palette)

## Layout Posture
- Warm cream page background, white card surfaces
- Deep forest green for headings and primary text
- Pastel teal as primary accent for buttons, links, interactive elements
- Pastel yellow for secondary highlights and badge backgrounds
- Pastel green for success states and feature card accents
- Rounded corners (12–16px) for cards and buttons
- Subtle shadows on interactive elements
- Generous whitespace, centered layout
- Smooth scroll animations (fade-in-up on scroll)
- Sticky frosted navigation bar
